# 手测冒烟清单

## 1. 主窗口启动

- 启动应用后，主窗口可以正常显示
- 主窗口标题栏、导航栏、聊天区、右侧工作台都可见
- 没有明显乱码文案

## 2. 悬浮球显示

- 主窗口最小化到悬浮球后，悬浮球可见
- 悬浮球不会跑到屏幕外
- 悬浮球在多次点击后不闪烁、不明显位移

## 3. 悬浮球菜单

- 左键点击悬浮球可以打开菜单
- 再点一次可关闭菜单
- 点击外部区域后菜单关闭
- 菜单打开时，悬浮球本体不位移
- 菜单项中文显示正常

## 4. 悬浮球菜单功能

- `聊天入口` 子菜单可以展开
- 外部聊天入口点击后能正常打开对应页面
- `界面外观` 子菜单可以展开
- `鼠标随航` 勾选状态可切换
- `设置` 点击后能拉起主窗口设置页

## 5. 主聊天区

- 输入框占位符、工具按钮、发送按钮中文正常
- 输入文本后可以发送
- `Shift + Enter` 可以换行
- 发送中可以点击停止
- `新话题` 按钮在有会话时显示正常

## 6. 话题与工作台

- 右侧 `话题 / 任务 / 自动化 / 记忆` 标签可切换
- 话题搜索可输入
- 话题菜单可打开
- 删除未收藏话题、删除全部话题按钮文案正常

## 7. 助手与设置

- 左侧基础聊天、自定义助手列表正常显示
- 助手设置可进入
- 助手名称、描述、角色设定、默认模型可以编辑
- 头像选择面板可打开
- 工具权限、技能权限、记忆策略文案正常

## 8. 模型配置

- 设置页 `基本设置 / 模型配置` 中文正常
- 新增模型弹窗可打开
- 接口名称、接口地址、访问密钥、模型 ID、显示名称输入正常
- 测试连接、保存模型、删除模型按钮文案正常

## 9. 数据持久化

- 重启应用后，助手、话题、设置仍然保留
- Provider 配置仍可读取
- 聊天记录、记忆、自动化、任务历史仍可恢复

## 10. 回归风险点

- 主窗口不会在启动时被莫名隐藏
- 悬浮球可见但菜单打不开的问题不再出现
- 菜单打开即关闭的问题不再出现
- 打开菜单时悬浮球移位的问题不再出现
- token 估算 badge 不再乱码
