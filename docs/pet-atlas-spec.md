# 桌宠大图集规格

## 目标

下一版桌宠图集不再只按技术状态设计动作，而是参考雪纳瑞桌宠动作板，按真实桌面使用场景组织动作。

规格：
- 图集尺寸：`8 x 16`
- 单元尺寸：`192 x 208`
- 总尺寸：`1536 x 3328`
- 每行一个动作
- 不足 8 帧的部分透明留空

## 第一版 8x16 动作

1. `idle-front`：正面待机。
2. `idle-side`：侧面待机。
3. `idle-back`：背面待机。
4. `walk`：行走循环。
5. `run`：奔跑循环。
6. `happy`：开心表情。
7. `blink`：自然眨眼。
8. `surprised`：惊讶反应。
9. `sad`：委屈难过。
10. `curious`：歪头好奇。
11. `greet`：打招呼。
12. `clingy`：撒娇互动。
13. `stretch`：伸懒腰。
14. `sleep`：睡觉。
15. `reading`：看书思考。
16. `computer`：电脑前工作。

## 前端场景映射

- 默认空闲：`idle-front`、`idle-side`、`blink`、`curious`。
- 打开菜单：`greet`。
- 打开查询：`curious`。
- 等待输入：`idle-front`。
- 查询处理中：`computer`。
- 思考中：`reading`。
- 回复成功：`happy`。
- 回复失败：`sad`。
- 拖动：`run`。
- 随屏移动：`walk` 或 `run`。

## 生成约束

- 所有动作必须保持同一只雪纳瑞的脸型、耳朵、项圈、吊牌和毛色一致。
- 不允许速度线、地面阴影、独立爱心符号、问号符号、文字或 UI 元素。
- 允许通过姿态、头部角度、眼神、耳朵、尾巴和前爪表达动作。
- 行图源文件可以是较大尺寸，但进入 `decoded` 前必须标准化为 `frames * 192 x 208`。
