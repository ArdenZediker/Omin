# Omni 界面可视化收尾清单

> 说明：
> - 这份清单只覆盖“用户在界面上能直接看到、操作到”的能力。
> - 完成一项就勾选一项，并同步更新状态记录。
> - 后续每完成一个任务，必须立即更新本文件，不跳过、不延后。
